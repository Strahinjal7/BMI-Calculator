Features

Accepts weight (kg) and height (m) from the user

Validates that values are numeric and greater than zero

Calculates BMI using the standard formula

Returns accurate health category

Option to repeat the calculator

User-friendly and beginner-friendly structure

How to Run

Install Python

Run the script in your terminal or IDE

Follow the on-screen instructions

Example Output

Your BMI is: 23.4
Your category is: Normal weight
